# Dinesh Kadel | Fullstack Developer Portfolio

A responsive, animated portfolio website built with **React** and **Tailwind CSS**, showcasing skills, experience, projects, and contact details.

## Features

- Responsive on desktop and mobile
- Dark/light theme toggle
- Smooth section reveal animations
- Anchor navigation menu: Home, About, Education, Skills, Projects, Contact
- Contact section with email, LinkedIn, GitHub, phone
- Vite build and deploy ready

## Tech Stack

- React + Vite
- Tailwind CSS
- Font Awesome icons
- Modern JavaScript (ES6+)

## Project Structure

- `src/`
  - `main.jsx`, `App.jsx`
  - `index.css`
  - `components/`
    - `layout/` (Navbar, Footer)
    - `sections/` (Hero, About, Education, Skills, Projects, Contact)
    - `ui/` (Button, Reveal, SectionHeader)
  - `data/` (`portfolioData.js`)
- `public/` (static assets)
- `package.json`, `vite.config.js`, `tailwind.config.js`

## Installation & Local Development

1. Clone the repository
   - `git clone https://github.com/dines-1/Dines-1-Personel_Portfolio.git`
2. Go into project directory
   - `cd react_portfolio`
3. Install dependencies
   - `npm install`
4. Start development server
   - `npm run dev`
5. Open in browser:
   - `http://localhost:5173`

## Build

- `npm run build`
- `npm run preview`

## Deployment

Compatible with:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service

##  Customize

- Update content in `src/components/sections/*`
- Update project data in `src/data/portfolioData.js`
- Control section layout in `src/App.jsx`
